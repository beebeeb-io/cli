# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Beebeeb, please report it responsibly.

**Email:** security@beebeeb.io

**PGP Key Fingerprint:** (to be published)

We aim to acknowledge all reports within **48 hours** and will work with you to understand and address the issue promptly.

## Scope

The following are in scope for security reports:

- All Beebeeb client applications
- The Beebeeb API
